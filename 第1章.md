## 第1章：はじめに

---

### 1-1. 目的

本フレームワーク「テンポリウム」は、時間旅行に関する条件を網羅的に定義し、タイムパラドックスの発生条件と成立可否を判定するための独自体系である。

従来のタイムパラドックス議論は、既存理論（ノヴィコフ自己無撞着性原理、多世界解釈、ブロック宇宙論など）に依存し、それぞれの理論が正しいという前提のもとで「パラドックスは回避される」と結論づけてきた。

しかし、これらの理論はいずれも仮説であり、どれが正しいかは確定していない。

テンポリウムは、既存理論に依存せず、時間旅行の「条件」を網羅的に定義し、その条件の組み合わせによって矛盾が発生するか整合が保たれるかを判定する。

これにより、以下の目的を達成する。

| 目的      | 内容                                  |
| ------- | ----------------------------------- |
| 検証基盤の提供 | タイムパラドックスの発生メカニズムを体系的に検証するための判定基盤   |
| 理論評価の実現 | 既存のタイムパラドックス理論が妥当かどうかを評価するメタフレームワーク |
| 汎用的な適用  | 理論検証、フィクション創作、思考実験など多様な用途への適用       |

---

### 1-2. 適用範囲

テンポリウムは以下の用途に適用可能である。

|用途|説明|
|---|---|
|理論検証|既存のタイムパラドックス理論が、どの条件下で成立するかを評価する|
|フィクション創作|時間旅行を扱う物語の整合性をチェックする|
|思考実験|時間旅行に関する哲学的・論理的な議論の土台を提供する|
|行動指針|時間旅行が実現した場合のパラドックス回避行動を設計する|

---

### 1-3. 前提条件

本フレームワークは以下の前提に基づいて構築されている。

| 前提      | 内容                                            |
| ------- | --------------------------------------------- |
| 時間旅行の実現 | 時間旅行は技術的に実現しているものと仮定する                        |
| 理論非前提   | 特定の理論を前提とせず、複数の理論が想定するモデルを条件として並置する理論中立の体系である |
| 条件ベース判定 | パラドックスの成否は「条件の組み合わせ」によって判定される                 |
| 網羅性     | 時間旅行に関するあらゆる条件を7層55用語で網羅する（付録含め全65用語）         |
